Analyse des Modèles

RF Random Forest

Le modèle Random Forest s'est avéré être le plus performant avec un score R² de 0.862, indiquant qu'il explique 86.2% de la variance dans les données de consommation énergétique. Ce modèle utilise un ensemble d'arbres de décision pour capturer les relations non-linéaires complexes entre les variables météorologiques et temporelles.

Points forts :

  • Excellente gestion des relations non-linéaires
  • Robustesse face aux valeurs aberrantes
  • Capture efficace des interactions entre variables
  • Erreur moyenne absolue (MAE) de seulement 2650 Wh

GB Gradient Boosting

Le modèle Gradient Boosting arrive en deuxième position avec un score R² de 0.845. Il construit séquentiellement des arbres de décision, chacun corrigeant les erreurs du précédent.

Points forts :

  • Précision légèrement inférieure au Random Forest mais toujours excellente
  • Meilleure capacité à capturer les tendances subtiles
  • Bonne généralisation sur les données de test

RR Ridge Regression

La régression Ridge a obtenu un score R² de 0.725. Ce modèle linéaire avec régularisation L2 aide à réduire le surapprentissage en pénalisant les coefficients élevés.

Points forts :

  • Simple et rapide à entraîner (0.4 secondes)
  • Légèrement plus performant que la régression linéaire standard
  • Interprétabilité des coefficients

LR Régression Linéaire

La régression linéaire simple a obtenu un score R² de 0.724. Ce modèle de base suppose une relation linéaire entre les variables indépendantes et la consommation énergétique.

Points forts :

  • Modèle le plus simple et le plus rapide à entraîner (0.3 secondes)
  • Facilement interprétable
  • Performance décente pour un modèle aussi simple

Conclusion

Le modèle Random Forest est clairement supérieur pour cette tâche de prédiction de consommation énergétique, avec une erreur quadratique moyenne (RMSE) significativement plus basse et un coefficient de détermination (R²) plus élevé que les autres modèles. Sa capacité à capturer les relations complexes entre les variables météorologiques et temporelles en fait le choix optimal pour ce projet.

Pour les cas où la vitesse d'inférence est critique ou lorsque l'interprétabilité est prioritaire, la régression Ridge peut être une alternative acceptable, bien que moins précise.
