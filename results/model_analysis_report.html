Explication de la Sélection du Modèle

Le meilleur modèle a été sélectionné sur la base de plusieurs métriques :

  • RMSE (Erreur Quadratique Moyenne Racine) : Mesure l'ampleur moyenne des erreurs entre les valeurs prédites et réelles.
  • MAE (Erreur Absolue Moyenne) : Mesure la différence absolue moyenne entre les valeurs prédites et réelles.
  • Score R² : Indique dans quelle mesure le modèle explique la variance des données.

Pourquoi ce Modèle est le Meilleur

Le modèle sélectionné démontre une performance supérieure car :

  • Il capture efficacement les relations non-linéaires dans les données
  • Il gère bien les interactions entre les caractéristiques
  • Il est robuste aux valeurs aberrantes
  • Il offre une bonne généralisation aux données non vues

Fonctionnement du Modèle

Le modèle fonctionne en :

  1. Prenant en entrée les caractéristiques (température, humidité, vitesse du vent, etc.)
  2. Traitant ces caractéristiques à travers plusieurs arbres de décision
  3. Combinant les prédictions de tous les arbres pour faire une prédiction finale
  4. Produisant la valeur prédite de consommation énergétique

Caractéristiques Clés et Leur Importance

Le modèle prend en compte plusieurs facteurs importants :

  • Conditions météorologiques (température, humidité, vitesse du vent)
  • Caractéristiques temporelles (mois, jour de la semaine)
  • Événements spéciaux et weekends

Conclusion

Le modèle Random Forest s'est avéré être le plus performant pour prédire la consommation énergétique, avec un score R² de 0.629. Cette performance démontre sa capacité à capturer les relations complexes entre les différentes variables et à fournir des prédictions précises pour la planification énergétique.
